Wi-Fly is a remote control app for X-Plane by Laminar Research (unaffiliated). In order to use this app, you must have a copy of X-Plane on your desktop computer and install the Wi-Fly plugin. Your iPhone or iPod Touch must be connected via Wi-Fi to the same network as your computer.

To install the Wi-Fly plugin, point your desktop browser to http://danieldickison.com/wifly and follow the instructions under Downloads.

Once you have the plugin installed in X-Plane, you should see the following window when you start X-Plane:

On the remote, you must enter one of the Host IP(s) listed in the plugin window, as well as the Server Port if it differs. If there is more than one IP address listed, try each if the first one doesn't work.

Once the correct IP and port are entered, you should see the bars in the plugin window moving as you tilt your iPhone or iPod Touch.

Feel free to contact support at danieldickison+wifly@gmail.com if you need any help
